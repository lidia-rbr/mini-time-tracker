\# Mini Time Tracker ⏱️



A lightweight, browser-based time tracking tool to quickly record time spent on different projects.  

No backend, no database — everything runs locally in your browser.



\## Features

\- Add multiple projects

\- Start/stop timers

\- See total active time

\- Dark/light mode toggle

\- Live clock for your timezone



\## Tech Stack

\- HTML

\- CSS

\- Vanilla JavaScript



Perfect small utility for freelancers or anyone who wants a minimal time tracker!
